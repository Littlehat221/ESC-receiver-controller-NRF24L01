# ESC-receiver-controller-NRF24L01
Receiver and controller(transmitter) for electronic speed controllers. Designed using NRF24L01 transceiver

Specs
- 4 PWM signals plus one digital signal
- Utilizes:
  - 2 joysticks
  - 4 tactile buttons
  - 2 potentiometers
  - 2 toggle swiches
